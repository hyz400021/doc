### 1. 拉取最新代码
```bash
git pull
```
### 2. 添加所有更改过的文件
```bash
git add .
```
### 3. 提交更改
```bash
git commit -a -m "mark in it"
```
### 4. 推送到远程仓库
```bash
git push origin main
```

## 判断本地分支与远程分支是否一致
```bash
git branch -vv
```

## 判断本地分支与远程的这个分支哪个更新一些
```bash
git status
```
